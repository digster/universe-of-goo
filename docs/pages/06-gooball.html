06 · Full goo ball

A goo ball is just a particle with a type and a pair of eyes — what makes the game click is the attachment rule: when a free ball meets a structure, which balls does it bond to?

Anatomy of a goo ball

In src/game/gooball.js: 

gooball = {
  id, type, color,
  attached,           // true if part of the structure, false if wandering
  pinned,             // true for `fixed` anchors (invMass = 0)
  particle,           // the underlying Verlet particle
  strands,            // constraints this ball participates in
  gazeX, gazeY,       // where the eyes look (towards cursor / motion)
}

Each goo type picks a colour, a max strand count, and per- step behaviour. The four shipped types are basic, balloon (lifts), sticky (high stiffness, 3 strands), and fixed (invMass = 0, never moves).

The strand-attachment rule

When a free ball is released near attached balls, we ask:

  1. Which attached balls are within radius R of me?
  2. Sort by distance, closest first.
  3. Walk down the list; accept a candidate only if the straight-line segment to it doesn't pass through static geometry (line-of-sight).
  4. Skip candidates that have already hit their type's maxStrands cap.
  5. Take the first K that pass — K is the new ball's own maxStrands.

This is implemented in src/game/strandRules.js. Line-of-sight uses a straightforward segment–segment intersection check against each static collider — cheap enough at the handful-of- walls scale of each level. 

candidates = balls.filter(b => b.attached && dist(b, me) < R)
            .sort(by distance)
            .filter(lineOfSight)
            .filter(b => b.strandCount < b.type.maxStrands)
attach to first K of candidates

Live demo

A seed tower sits at the centre. Click anywhere on the canvas to drop a new basic ball. It'll form up to K strands (your slider) with attached balls inside radius R, respecting line-of-sight. Drop balls through the gap in the tower to watch the line-of-sight check reject candidates on the far side.

At K = 1 the tower grows a hairy tree: every new ball makes a single strand to its closest neighbour. At K = 3 it becomes tightly triangulated. The game ships with basic.maxStrands = 2 as the balance point.
